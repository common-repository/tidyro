A user friendly plugin that helps you send articles to the tidy Network with no effort. 

== Description ==

Send articles to the tidy Network with just the press of a button. 
After your publish your articles, just press the "Submit to tidy.ro" button and you are done. 
The articles will then be sent to our reviewers. 
You will also be able to track the traffic, you send or receive to the network.


== Installation ==

1. Upload `tidy` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get your Site Secret Key from http://tidy.ro/websites and enter it in the tidy plugin and click Save
4.Start posting to the tidy Network

== Frequently Asked Questions ==

= Where do we get the Secret Key ? =

http://tidy.ro/websites - The Secret Key will appear in the list after the website has been approved, also it is contained in the email when the site is approved.

=  How do I add the widget to my blog? =

* Just select the category you want our widdget to appear on and copy/paste the script code you were provided on tidy.ro
* Or you can copy/paste the javascript code anywhere in the theme or posts.

= Can I see me statistics live? =

Yes you can.

= Do I need to login to tidy.ro to post my articles? =

No, you don't. After you have obtained your Secret Key and configured the plugin properly, you can do most of the stuff directly from your wordpress admin.
